Claude Week

Five days of hands-on challenges, a live build competition, and awards celebrating different kinds of excellence with Claude Code.

How it works

Each morning a daily challenge drops in Slack. Set your own timer, do the work, and post your result in the thread before EOD your timezone. Challenges are designed to be short — the constraint is the point. By Friday you'll have five real outputs and a much sharper sense of what Claude Code can do in a focused window.

Points: 10 base per challenge + 5 bonus for nailing the extra credit. Scores are tracked on the leaderboard and feed into the awards at Friday's closing ceremony.
Daily Challenges
Monday The Refactor Sprint

Pick a messy function from your codebase. Use Claude Code to refactor it for readability and performance and add JSDoc. Post your before/after diff in the Slack thread.

Timer: 15 min The time limit isn't a hard rule — it's a prompt to show how far you can get quickly. Stop when it goes off and post what you have.
+5 bonus If it also generates passing tests.
Tip Drop the whole file and let Claude find the worst offender.
Tuesday The Bug Hunt

Paste a broken snippet or failing test into Claude Code. Get it working and post your fix along with a one-line explanation of the root cause.

Timer: 10 min Short on purpose — debugging with Claude Code is fast when you let it lead.
+5 bonus If Claude's root cause explanation is clear enough to drop straight into a PR comment.
Tip Ask Claude to explain why before asking it to fix.
Wednesday The Docs Writer

Pick any undocumented function or module. Use Claude Code to generate a README section or inline docs. Post the output.

Timer: 10 min
+5 bonus If it includes a working usage example.
Tip Tell Claude who the audience is — "write this for a new engineer joining the team."
Thursday The CLI Wizard

Build a small CLI tool or script using Claude Code — anything that saves you or your team real time. It must run. Share what it does and show it working.

Timer: 20 min A little more room — you're building something from scratch.
+5 bonus For creative use of Claude Code slash commands.
Tip Start with "I want a script that…" and iterate from there.
Friday The Prompt Surgeon

Find a prompt that gave you a mediocre result this week. Iterate on it live until Claude Code nails it. Document every iteration and post the full arc.

Timer: 15 min
+5 bonus If the final result is genuinely impressive.
Tip The messier your starting prompt, the better the story.
Build-Off Finals — Friday midday

Two hours. Build anything. Make it work.

A live synchronous session held midday Friday when everyone is available. Solo or pairs. No constraints on what you build — the only rule is that it has to run. This is your chance to put the whole week together and make something real.
